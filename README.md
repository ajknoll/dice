dice
====

Simple CLI dice roller for tabletop gaming.
You can roll multiple pools of dice at once, with arbitrarily long strings of modifiers (+x, -x, *x, /x) attached to each pool.

Examples:
---------

> python dice.py 1d6
>            ... 1d20+14 1d8-1
>            ... 1d3+2+4-3+1
>            ... 1d12*2
